Encargo 05: Impresión 3D

En este post veremos desde un proceso de simulación como se efectúa una impresión 3D. La Impresión 3D es una avance muy importante de tecnologías de fabricación por adición donde un objeto tridimensional es creado mediante la superposición de capas sucesivas de material.

En este caso usaremos el software “Ultimaker Cura”

Luego de descargar el programa debemos seleccionar la impresora que vamos a utilizar. Dado que vamos a trabajar offline, vamos a “ADD A NON-NETWORKED PRINTER” y escogemos la Creality Ender 3 que se ve en la siguiente imagen.

Después debemos elegir el material con el cual vamos a imprimir. Para esto vamos al menu superior donde dice Generic PLA. En estas opciones se debe elegir el material y el tamaño del cabezal, para este caso usaremos de material “eSUN” y luego “PLA+”, el color depende del contraste. Para el cabezal será de 0.4mm.

Para comenzar a trabajar debemos abrir nuestro archivo “STL”.

Si nuestro objeto a imprimir tiene unas zonas de color rojo significa que tiene que necesitar soporte. Entonces debemos estructurar el objeto para que ocupe el menor espacio como soporte. Para cambiar los valores del material de soporte, como el ángulo o el lugar debemos seleccionar el menu la opcion “NOT SUPPORTED” y luego “SUPPORT”. También es importante cambiar los mm en la opción “QUALITY” a 0.3mm.

Luego de realizar los ajusten en “NOT SUPPORTED” presionamos la opcion SLICE y el programa nos deja ver como deberia crearse nuestro objeto.

El programa nos especifica:

  • Tiempo impresión
  • Material
  • Cantidad de capas

Para modificar el trabajo y hacerlo mas eficiente, se puede cambiar el ángulo de la figura logrando que se generen menos capas. También otra opcion es bajar la densidad del relleno de nuestra figura, para eso en la opción “INFILL” bajamos su densidad a 6%.

Los resultados fueron disminuir las capas, material y tiempo de trabajo.

Para tener más informacion sobre el color de las lineas de nuestras capas en la opcion del menu COLOR SCHEME se nos informa en detalle que significa cada color.
