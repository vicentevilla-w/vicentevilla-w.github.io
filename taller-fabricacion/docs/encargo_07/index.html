Encargo 07: Complementario

En este post veremos como insertar un iFrame de Fusion 360 en nuestra plataforma web mediante Markdown. En este caso usaremos un prototipo de un modelo 3D, realizado en el post anterior.

Paso 1

  1. Guardar archivo:

Luego de tener nuestro objeto listo en el programa “FUSION 360” debemos guardar el archivo en Admin Proyect nuestra carpeta online. De esta manera queda almacenado en la nube de “Autodesk”

Paso 2

  1. Ingresar a “Autodesk”

Ingresamos a la pagina: “http://www.myhub.autodesk360.com”

Se nos va a solicitar nuestro usuario, contraseña y a veces se nos envía un cógio a nuestro numero de teléfono para confirmar el usuario. Al entrar nos aparece un menú donde estan todos nuestros projectos.

Paso 3

  1. Copiar link:

Luego para copiar el archivo hacia nuestro iframe, tenemos que copiar el link de autodesk con nuestro usuario. Para eso seleccionamos el archivo que queremos postear y para encontrar el link vamos a la opción Share (ícono señalizado en un círculo rojo), y dentro de esta misma vamos hacia Embed y seleccionamos el link “HTTP”

https://alumnos4811.autodesk360.com/shares/public/SH9285eQTcf875d3c53937f97fd76f06c444?mode=embed

Paso 4

  1. Usar iFrame:

Al tener el link del archivo en la nube usamos el código markdown para ingresar los iFrame.

Entonces dentro de las comillas (" “) pegamos el link seleccionado.

De esta manera logramos usar de forma correcta el iFrame de Fusion 360 para postear cualquier trabajo que realizado en el programa.
